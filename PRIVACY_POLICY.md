# Privacy Policy for HomeKitTV

**Effective Date**: December 10, 2025
**Last Updated**: December 10, 2025

## Overview

HomeKitTV is a tvOS application designed to control HomeKit-enabled smart home devices from your Apple TV. This privacy policy explains how HomeKitTV handles your information.

## Information Collection and Use

### What We DO NOT Collect

HomeKitTV **does not collect, store, or transmit** any of the following:
- Personal information
- HomeKit device data
- Usage analytics
- Crash reports
- Location data
- Network information
- Any identifiable information

### What Stays on Your Device

All HomeKit data remains entirely on your local device and network:
- Your home configuration
- Device names and settings
- Scene configurations
- Room assignments
- Automation settings

## HomeKit Data Usage

### Why We Access HomeKit

HomeKitTV requires HomeKit access solely to:
- Display your HomeKit accessories
- Control device states (on/off, brightness, temperature)
- Execute scenes
- Organize devices by room
- Show real-time device status

### How HomeKit Data is Handled

- **Local Only**: All HomeKit interactions happen locally on your device
- **No Cloud Storage**: We do not sync or store HomeKit data to any server
- **No Third Parties**: HomeKit data is never shared with third parties
- **Apple's Control**: HomeKit data is managed entirely by Apple's HomeKit framework
- **Your Control**: You can revoke HomeKit access anytime in Settings

## Data Storage

HomeKitTV stores only:
- **User preferences** (locally on device)
  - Display settings
  - Favorite accessories
  - UI customizations
- **No personal data** is stored

All storage is local using Apple's standard UserDefaults and is sandboxed to HomeKitTV only.

## Network Usage

HomeKitTV uses your local network to:
- Communicate with HomeKit hub (Apple TV, HomePod, or iPad)
- Send control commands to accessories
- Receive status updates from devices

**No internet connection is required** for normal operation. HomeKitTV does not:
- Send data to external servers
- Make API calls to remote services
- Track or log network activity
- Use analytics services

## Permissions

### Required Permissions

**HomeKit Access** (Required)
- Purpose: Control your HomeKit accessories
- Usage: Read device states, send control commands, execute scenes
- Revocation: Settings → Privacy → HomeKit → HomeKitTV

**Local Network** (Automatic)
- Purpose: Communicate with HomeKit devices
- Usage: Local network communication only
- No internet access required

## Third-Party Services

HomeKitTV does not use any third-party services, including:
- Analytics platforms
- Advertising networks
- Crash reporting services
- Cloud storage providers
- Social media integrations

## Children's Privacy

HomeKitTV does not knowingly collect information from children under 13. The app is rated 4+ and contains no content inappropriate for children. No personal information is collected from any user, regardless of age.

## Security

While HomeKitTV does not collect data, we implement security best practices:
- **Sandboxed**: App runs in Apple's sandbox environment
- **Code Signing**: App is signed with Apple Developer certificate
- **HomeKit Security**: Leverages Apple's HomeKit encryption and security
- **No External Access**: No data leaves your local network

## Your Rights

As a user, you have the right to:
- **Access**: View all your HomeKit data through Apple's Home app
- **Control**: Manage HomeKit permissions in Settings
- **Delete**: Remove the app to delete all local preferences
- **Opt-Out**: Revoke HomeKit access anytime
